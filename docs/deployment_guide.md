# Guide de Déploiement (Deployment Guide)

## Cloud Run / Container
1. Assurez-vous que Docker est installé.
2. Build : `npm run build`.
3. Start : `npm start`.
4. Le port utilisé est le `3000`.

## PWA
Le manifest et le Service Worker sont automatiquement configurés et servis à la racine.
L'image `/logo-512.png` sert de source pour toutes les icônes.
